# Client Portal UI Kit

## Overview
The client-facing surface of the Novum platform. Clients (`role: 'cliente'`) access a personal production board, script viewer, and delivery calendar.

## Screens
- **index.html** — Kanban board (default landing, `/kanban`)
- Views: Kanban, Ver Guion (script table), Calendario mensual

## Navigation
Top header: logo left, nav links (Mi Kanban / Calendario), client name + logout right.

## Design Notes
- Light surface: `#F5F5F2` page, `#FFFFFF` cards
- Kanban columns color-coded by stage
- No sidebar — minimalist top nav only
